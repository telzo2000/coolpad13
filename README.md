# coolpad13

coolpad13 is QMK_Firmware powered, bluetooth enabled, 5 key macropad keyboard.
<br>
coolpad13はQMK_Firmwareで動き、ブルートゥース対応で、５キーのマクロパッドキーボードです。
<br>
